#### Unit 5: What's this?
| 类别 | 内容 |
|------|------|
| 核心词汇 | ladybird, cicada, butterfly, dragonfly, cute |
| 核心句型 | What's this, Liu Tao? — It's a cicada. / Look at the ladybird. — It's cute. |

| 字母学习 | Oo, Pp, Qq |

| 类别 | 内容 |
|------|------|
| 核心词汇 | run, jump, hop, walk |
| 核心句型 | Are you ready? — Yes! / Well done! / One by one! / Let's do it. / Hop, hop, hop. |
| 交际功能 | 课堂指令与动作反应，节奏训练，肢体指令与语音结合 |
| 字母学习 | Rr, Ss, Tt |

#### Unit 7: What's that?
| 类别 | 内容 |
|------|------|
| 核心词汇 | cow, duck, lamb, pig, baa, quack |
| 核心句型 | What's that? — It's a lamb. / What's this? — It's a cow. / Shh! |
| 交际功能 | 区分空间远近进行问答（近处What's this? / 远处What's that?），农场动物认知 |
| 字母学习 | Uu, Vv, Ww |

#### Unit 8: What's in your bag?

| 字母学习 | Xx, Yy, Zz |

## 七、使用说明

1. **数据录入参考**：家长录入考试成绩和错题时，可对照本表找到对应的知识点
2. **AI分析基准**：AI分析学生薄弱点时，以此知识点体系为依据，不超纲
3. **练习推荐**：根据学生薄弱点映射表，推荐对应知识点的练习题
4. **进度跟踪**：对照教学进度表，判断学生是否跟上了学校进度
5. **持续维护**：每学期初根据最新教材版本更新此文档

> ⚠️ 注意：此文档基于2024-2025学年江苏地区使用的教材版本整理。
> 教材如有更新，请及时对照最新版本修订。
